
Инструкция по использованию:

1. Откройте приложение
1. Нажмите кнопку "Выбрать файл" и выберите на своем устройстве список участников в формате JSON или CSV
3. Нажмите кнопку "Загрузить". После загрузки участники появятся в правой части экрана
4. Нажмите кнопку "Крутить", чтобы запустить колесо фортуны. После остановки приложение покажет участника и выпавший приз
5. Повторяйте розыгрыш, пока все участники не получат призы.
6. Нажмите кнопку "История", чтобы скачать результаты розыгрыша в файл `history.json`.
7. Нажмите кнопку "Сброс", если хотите очистить список участников и историю

Поддерживаемые файлы:

- JSON: массив участников или объект с полем `participants`;
- CSV: список имен

Выходной файл с историей розыгрыша history.json
